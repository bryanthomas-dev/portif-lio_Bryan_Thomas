# 🏁 SM1 - A Corrida do Prompt

## 📝 Descrição do Projeto

Este projeto marca o primeiro contato prático com Engenharia de Prompt, explorando como a forma de se comunicar com modelos de linguagem impacta diretamente na qualidade e precisão das respostas geradas.

Desenvolvido como parte da disciplina de **Engenharia de Prompt e Aplicações em IA (2026.1)**, o projeto consistiu em testar diferentes estruturas de prompts — variando clareza, contexto, tom e especificidade — para analisar como cada variação influencia o comportamento da IA. O objetivo foi desenvolver a habilidade de "conversar estrategicamente" com modelos de linguagem para obter resultados mais úteis e precisos.

## 🚀 Tecnologias Utilizadas

* **Ferramentas de IA:** Claude, ChatGPT, Qwen, Gemini
* **Plataforma:** Navegador web (interfaces dos modelos)
* **Linguagem:** Português / Prompts em linguagem natural

## 📊 Resultados e Aprendizados

* **Compreensão prática** de como o contexto e a especificidade de um prompt afetam o resultado gerado pela IA.
* **Comparativo entre modelos:** diferentes IAs responderam de formas distintas ao mesmo prompt, revelando pontos fortes e fracos de cada uma.
* **Desenvolvimento da habilidade** de estruturar prompts claros, objetivos e eficazes para diferentes tarefas.

## 🔧 Como Reproduzir

1. Acesse qualquer ferramenta de IA (Claude, ChatGPT, Gemini etc.).
2. Consulte os prompts documentados nesta pasta.
3. Execute e compare os resultados entre diferentes modelos.
